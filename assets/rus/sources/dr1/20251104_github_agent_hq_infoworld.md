# GitHub Agent HQ — диспетчерская для кодовых ИИ-агентов

> **Scrape context:** материал InfoWorld (29 октября 2025) об официальном запуске Agent HQ и реакции аналитиков.
>
> **Source:** [InfoWorld](https://www.infoworld.com/article/4080888/github-launches-agent-hq-to-bring-order-to-ai-powered-coding.html)

## Что такое Agent HQ
- Единый центр управления внутри GitHub, который оркестрирует агентов от OpenAI, Anthropic, Google и др.
- Доступен на GitHub.com, в VS Code, CLI и мобильном приложении; даёт миссион-контроль для всех задач, выполняемых ИИ.

## Функции для enterprise
- **Плоскость управления:** политика доступа к агентам, аудит, дашборды влияния ИИ на код.
- **Agentic code review:** новый формат ревью с сохранением следов и метрик качества.
- **Кастомные агенты:** в VS Code можно описать конфиг (правила код-стайла, ограничения, точки эскалации) и получить собственного агента.
- **Интеграции:** Slack, Teams, Jira, Azure Boards и др. тянут события из Agent HQ в привычные каналы.

## Зачем это GitHub
- Корпорации одновременно тестируют нескольких вендоров ИИ; GitHub хочет стать «оркестратором», чтобы разработчики не прыгали между отдельными приложениями.
- По данным IDC, лишь ~16% времени девелоперы тратят на создание нового кода, остальное забирает вспомогательная рутина — агентные инструменты должны вернуть эффективность.

## Мнение аналитиков
- Шаратх Шринивасамурти (IDC) считает, что Agent HQ может стать «центральным источником» для всех агентных инструментов и сократить контекстные переключения.
