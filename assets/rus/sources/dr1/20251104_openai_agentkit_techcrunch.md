>
> **Source:** [TechCrunch](https://www.techcrunch.com/2025/10/06/openai-launches-agentkit-to-help-developers-build-and-ship-ai-agents/) · 11:00 AM PDT, 06.10.2025

## Что анонсировали
Sam Altman представил AgentKit как «полный набор блоков» для переноса агентов из прототипов в продакшен с минимальными трениями. Запуск идёт в связке с новой функцией построения приложений прямо в ChatGPT (800 млн WAU) и должен удержать OpenAI впереди конкурентов, предлагая разработчикам сквозной пайплайн.

## Состав AgentKit
- **Agent Builder.** Канва-подобный визуальный дизайнер поверх Responses API для оркестрации логики без ручного кода.
- **ChatKit.** Встраиваемые чат-компоненты, позволяющие перенести «openai-уровень» UX и фирменные флоу в существующие продукты.
- **Evals for Agents.** Трассировочные метрики, датасеты для подсистем, автоматический подбор подсказок и запуск оценок на внешних моделях из единого интерфейса.
- **Реестр коннекторов + админ-панель.** Каталог интеграций с внутренними и сторонними системами с контролем прав, что важно для enterprise.

## Доказательство зрелости
Инженер OpenAI Кристина Хуанг собрала два агента и полный workflow за <8 минут прямо на сцене. По словам Альтмана, несколько партнёров уже масштабируют продакшен-агентов на AgentKit, то есть релиз ориентирован на реальные внедрения.

## Почему это важно для Infra App
- Подтверждает, что набор «визуальный билдер + eval-инструменты + каталог коннекторов» — новый базовый стандарт, которого ждут клиенты в DR-треках.
- Подсвечивает необходимость прозрачных админ-интерфейсов и политик, когда продаём compliant-агентов российским облачным провайдерам.
