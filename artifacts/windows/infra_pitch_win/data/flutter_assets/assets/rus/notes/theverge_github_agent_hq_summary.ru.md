# GitHub Agent HQ — выдержки из The Verge / Sources

- **Стратегия Microsoft.** На GitHub Universe 2025 компания представила Agent HQ — интерфейс, куда могут встраиваться внешние кодовые ассистенты (OpenAI Codex, Anthropic Claude Code, Google, Cognition, xAI). Цель — оставить GitHub «домом разработки» даже в эпоху агентов.
- **Ставка на открытую экосистему.** Новый SVP Джаред Палмер называют GitHub платформой не только для людей, но и для агентов; COO Кайл Дайгл подчёркивает, что разработчик должен подключаться к GitHub и переносить контекст в любые инструменты.
- **Скорость как требование.** EVP CoreAI Джей Парикх руководит 10 тыс. человек и перестраивает стек Microsoft под агентные приложения. Он говорит, что люди будут тратить больше времени на спецификации, а «GPU и агенты» — на код.
- **Конкуренция.** Если GitHub не ускорится, разработчики уйдут к новым IDE (Cursor и др.), которые уже оркеструют рабочие процессы, а не просто дополняют код.
- **Важность Copilot.** Сатья Наделла напоминает, что именно ранний успех GitHub Copilot (поддержанный OpenAI) убедил Microsoft увеличить инвестиции в OpenAI с $1B до $10B.
- **Инсайт для нас.** Агентные «командные центры» становятся стандартом: нужно строить Console Copilot и MCP Pipeline Studio как аналогичный command center для инфраструктурного провайдера.
