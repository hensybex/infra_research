# Макроландшафт — AI-инфраструктура и агентные платформы (ноябрь 2025)

## Глобальные тренды
- **Agent Factory.** AWS GenAIIC Partner Agent Factory ускоряет совместную разработку production-ready агентов: партнёры приносят предметную экспертизу, AWS даёт методологию и канал дистрибуции через AWS Marketplace (`artifacts/scrape/20251105T134242Z_aws_partner_agent_factory.md`).
- **Экоcистемы для разработчиков.** GitHub Agent HQ превращает платформу в хаб мультивендорных кодинговых агентов (OpenAI, Anthropic, Google, Cognition, xAI), подчёркивая спрос на оркестрацию сквозь тулчейны (`assets/rus/notes/theverge_github_agent_hq_summary.ru.md`).
- **Дифференциация опыта.** Azure AI Foundry показывает эмоциональных и продуктивных агентов (Emotional Support Goose, Penultimate Penguin), акцентируя кастомизацию, guardrails и гибкость развёртывания от локального устройства до облака (`assets/rus/notes/azure_ai_foundry_future_agents_summary.ru.md`).

## Сигналы РФ/СНГ
- **Рост Selectel.** H1 2025: 8.9 млрд ₽ выручки (+46% г/г), 5.2 млрд ₽ EBITDA, 30.1 тыс. клиентов, агрессивные инвестиции в серверы и ЦОДы (`assets/rus/notes/akm_selectel_ifrs_summary.ru.md`). По данным TAdviser компания вкладывает 10 млрд ₽ в AI и продолжает выпуск облигаций (`assets/rus/notes/tadviser_selectel_highlights.ru.md`).
- **Рыночные бенчмарки.** Selectel занимает ~8.5% рынка облаков, лидирует в bare-metal аренде с долей 56% (`assets/rus/notes/wiki_selectel_summary.ru.md`), задавая планку для конкурентов уровня Timeweb.
- **Импульс marketplace.** Телекомы (МТС, Ростелеком) запускают суверенные витрины с AI-сервисами, что коррелирует с требованиями 152‑ФЗ к локализации данных и спросом на отечественные вычисления (`assets/rus/notes/mts_marketplace_news_summary.ru.md`).

## Контекст комплаенса
- Федеральный закон 152‑ФЗ требует хранить персональные данные граждан РФ внутри страны, уведомлять Роскомнадзор о трансграничных передачах и подтверждать «адекватность» юрисдикций. Штрафы достигают 1–18 млн ₽ и могут расти до 1–3% оборота при повторных нарушениях (`assets/rus/notes/wiki_data_localization_russia.ru.md`, `assets/rus/notes/DR2_5_compliance_checklist.ru.md`).

## Выводы для питча
- Дифференциация = мультиагентные, compliance-aware workflows, монетизируемые через marketplace.
- Компактной команде важно воспроизводить скорость AWS/GitHub при адаптации под российские ограничения и go-to-market.
