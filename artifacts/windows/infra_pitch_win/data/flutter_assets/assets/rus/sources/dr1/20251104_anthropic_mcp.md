# Model Context Protocol: открытый стандарт Anthropic

> **Scrape context:** анонс Anthropic (ноябрь 2025) об открытии Model Context Protocol (MCP) и запуске SDK/серверов для подключения ИИ к корпоративным данным.
>
> **Source:** [Anthropic](https://www.anthropic.com/news/model-context-protocol)

## Задача
- ИИ‑ассистенты до сих пор изолированы от корпоративных систем: для каждого хранилища или SaaS нужен отдельный коннектор.
- MCP — открытый JSON-RPC протокол, где сервер публикует инструменты/ресурсы один раз, а любой совместимый клиент (Claude, IDE, кастомный агент) автоматически их обнаруживает и вызывает по защищённому каналу.

## Что входит в релиз
- **Спецификация и SDK.** Репозитории на GitHub для клиентов и серверов.
- **Поддержка в Claude Desktop.** Локальные MCP‑серверы можно подключать прямо из десктопного приложения на базе Claude 3.5 Sonnet.
- **Каталог open-source серверов.** Готовые коннекторы к Google Drive, Slack, GitHub/Git, Postgres, Puppeteer и шаблоны для собственных интеграций.

## Первые пользователи
- Block и Apollo уже встроили MCP, чтобы их агенты получали контекст из разных систем.
- Разработчики инструментов (Zed, Replit, Codeium, Sourcegraph) добавляют MCP, чтобы кодовые агенты переносили контекст между IDE, репозиториями и документацией.
- Claude 3.5 Sonnet быстро генерирует MCP‑серверы, снижая порог входа для команд.

## Как стартовать
1. Установить Claude Desktop и подключить готовый сервер либо собрать собственный по quickstart.
2. Вклад в открытые репозитории приветствуется: можно публиковать новые коннекторы и расширения.
3. Клиенты Claude for Work уже тестируют локальные MCP‑серверы; Anthropic готовит инструменты для продакшен-развёртывания в масштабах всей организации.

## Комьюнити
Anthropic позиционирует MCP как открытый стандарт: чем больше команд делятся коннекторами и обратной связью, тем легче агентам переносить контекст между инструментами и данными.
